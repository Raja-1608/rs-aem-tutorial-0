# Velu Chits AEM Project

This repository contains the Velu Chits website built using the AEM Edge Delivery Services boilerplate.

## Environments

- Preview: `https://main--rs-aem-tutorial-0--raja-1608.aem.page/`  
- Live: `https://main--rs-aem-tutorial-0--raja-1608.aem.live/`

## Documentation

Before using the AEM boilerplate, we recommend going through the official documentation:

1. [Developer Tutorial](https://www.aem.live/developer/tutorial)  
2. [The Anatomy of a Project](https://www.aem.live/developer/anatomy-of-a-project)  
3. [Web Performance](https://www.aem.live/developer/keeping-it-100)  
4. [Markup, Sections, Blocks and Auto Blocking](https://www.aem.live/developer/markup-sections-blocks)

## Installation

1. Clone this repository.  
2. Open the project folder in VS Code.  
3. Make your content and style changes.  
4. Commit and push to GitHub to update the AEM environments.
